## 📌 About The Project

OSINT AI Tool is a Python-based cybersecurity reconnaissance and website security assessment framework designed to automate information gathering and basic security analysis of target domains.

It provides a clean terminal-based interface with multiple scanning modules, allowing security researchers, ethical hackers, and students to perform authorized reconnaissance efficiently.

> ⚠️ **Disclaimer:** This tool is created strictly for educational purposes and authorized security assessments only. Do not scan systems without proper permission.

---

## ✨ Features

### 🌐 Domain Intelligence
- Domain information lookup
- Domain registration details
- Name server enumeration

### 🔍 DNS Enumeration
- A Records
- AAAA Records
- MX Records
- NS Records
- TXT Records

### 🕵️ WHOIS Analysis
- Domain owner information
- Registration data
- Domain expiry details

### 🌍 IP Intelligence
- IP Address lookup
- Geolocation details
- ISP information
- ASN information

### 🔐 SSL Analysis
- SSL certificate information
- Certificate validity checking

### 🚪 Port Scanning
- Open port discovery
- Service identification

### 🛡️ Security Assessment
- HTTP Security Header analysis
- Missing security headers detection
- Basic website security checks

### 📄 Report Generation
Export scan results in:
- PDF Reports
- TXT Reports
- JSON Reports

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/OSINT-AI-Tool.git
```

### 2. Move into the Project Directory

```bash
cd OSINT-AI-Tool
```

### 3. Install Required Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Tool

```bash
python main.py
```
